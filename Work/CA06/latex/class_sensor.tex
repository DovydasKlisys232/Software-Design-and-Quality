\doxysubsection{Detailed Description}
Abstract base class for all sensor types. 

Provides interface for reading sensor data. Derived classes must implement the \doxylink{class_sensor_a8a21a3d413970d0e8a4c1853fa92c1b5}{read()} method for specific sensor types.
